# 动态内存分布malloc
void* malloc (asize_t size)
* 基本单位：一个字节
* 定义：可以通过malloc函数分配一块连续内存，最后返回申请成功后这块内容的首地址
* 功能：向内存**堆区**申请一块连续可用的空间，并返回起始地址
*        若开辟成功，返回起始地址
*        若开辟失败，则返回NULL指针。因此malloc返回值要做检查哦
*        返回值类型为void*，malloc不知道开辟空间的类型，使用时使用者决定
*        size为0，malloc的行为标准是未定义的，取决于编译器
````
#include<stdio.h>
#include<stdlib.h>
int main()
{
	int n = 5;
	int* p=(int*)malloc(n*sizeof(int));//20字节 ,强制转换  可以知道是5个int类型数据
	//malloc(20); 此写法不好，可读性较低,具体什么类型的数据不知道
	//assert(p != NULL);//断言
	if (p == NULL)
	{
		printf("申请失败\n");
	}
	//代码到这里说明成功了
	for (int i = 0; i < n; i++)
	{
		*(p + i) = i + 1;//解引用赋值
	}
	for (int i = 0; i < n; i++)
	{
		printf("%d\n", *(p + i));//若不赋值，则里面的值为随机/未定义的

	}



	return 0;
}

````
* 堆上的空间，在使用完成后，都需要程序员手动释放，回收内存
*            # 用free函数
*            void free (void* ptr)
*           可以进行内存回收释放，非动态开辟不可用free进行释放
*            若ptr为NULL指针，则free什么都不做
````
free(p);   //如果只是释放指针，则指针p为野指针   p=NULL
p = NULL;

````
