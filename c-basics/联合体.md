# 联合体
* 定义：由一个或多个成员组成，成员可以是不同的类型，但编译器只为最大的成员分配足够的内存空间。
* 特点：所有成员共用一个内存空间，故也称其为共用体.
*       大小至少为最大成员大小，但不一定正好
*       联合体也要保证内存对齐规则：当最大成员大小不是最大对齐数的整数倍时，需对齐到最大对齐数的整数倍
````
#include<stdio.h>
union Un1
{
	char c[5];
	int i;
};

union Un2
{
	short c[7];
	int i;
};
int main()
{

	printf("%d\n", sizeof(union Un1));//8
	printf("%d\n", sizeof(union Un2));//16


	return 0;
}
